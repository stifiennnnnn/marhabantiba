### CARA MEMBUKA WEB MARHABAN:

Buka halaman GitHub repository ini.
Klik tombol 'Code' berwarna hijau di bagian atas.
Pilih 'Download ZIP' dan tunggu hingga unduhan selesai.
Ekstrak file ZIP yang telah diunduh.
Masuk ke folder yang diekstrak dan buka folder 'marhabantiba'.
Cari file 'index.html' dan klik dua kali untuk membukanya di browser.
Marhaban akan terbuka di browser default perangkat Anda.

⚠ Pastikan perangkat terhubung dengan internet
Disarankan menggunakan Google Chrome dengan resolusi 1920 x 1080
